# data-engineering-fundamentals
Core data engineering concepts including data modeling, OLTP vs OLAP systems, star schemas, and data pipeline fundamentals with practical explanations.

## Topics Covered
- OLTP vs OLAP systems
- Data modeling fundamentals
- Star schemas and dimensional modeling
- Fact and dimension tables
- Slowly Changing Dimensions (SCD)

The goal of this repository is to demonstrate conceptual understanding, tradeoffs, and design thinking used in real-world data engineering systems.
